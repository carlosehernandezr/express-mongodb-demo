What does?

This make a POST call on [root-url]/api/create , add a new user on the mongodb and return a person object if is succesfull. If the data is incorrect or the user exist, will return an object of this type: {error: "Error message"}

A person example

{ username: "Jhon Doe",
email: "jhondoe@email.com",
gender: "masculine",
color: "blue" }

Create a person on the database

Fields with * are required and fields with ** are required and would be unique.

What does?

This make a GET call on [root-url]/api/persons and return a list of persons that match with the search. If there is not a match return void.

Example

If you want obtain the list of five persons with gender maculine and favorite color blue, you can make the next call:
[root-url]/api/persons?gender=masculine&color=blue&limit=5
if you introduce an username the call return a unique person, because username is unique for each person.
