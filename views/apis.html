APIs Examples

Example of API calls.

/api/hello

Test this endpoint

Return a simple JSON object with the message "Api say Hello!".

/api/calc

Test this endpoint

Return a simple calc into two numbers.

Need three query params:

Param Type
first Number
second Number
operation String

The operation param accept the next options: "sum", "substraction", "multiplication" and "division".

/api/whoami

Test this endpoint

Return the IP address, preferred languages (from header Accept-Language) and system infos (from header User-Agent) for my device.
